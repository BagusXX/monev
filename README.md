# Sistem Monitoring dan Pelaporan Kegiatan & Rapat

## Deskripsi

Sistem Monitoring dan Pelaporan ini merupakan aplikasi berbasis web yang dibangun menggunakan framework Laravel. 
Aplikasi ini digunakan untuk mengelola data kegiatan dan rapat, memantau laporan, serta mengatur data berbasis wilayah (daerah) dengan sistem persetujuan (approval) pengguna.

Sistem ini dilengkapi dengan fitur autentikasi, manajemen user, monitoring data, serta kontrol akses berbasis middleware.

---

## Fitur Utama

- Autentikasi pengguna (Login, Register, Reset Password)
- Sistem approval pengguna (Approved / Rejected / Admin)
- Middleware pembatasan akses pengguna
- Manajemen data Kegiatan
- Manajemen data Rapat
- Monitoring kegiatan dan rapat
- Manajemen wilayah berbasis Daerah
- CRUD User dan Wilayah
- Validasi menggunakan Form Request
- Seeder data Daerah

---
